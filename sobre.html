Motivo desta aplicação existir

Estou participando do Hiring Coders, que é um programa de educação e seleção de pessoas desenvolvedoras que desejam trabalhar nas principais empresas de tecnologia para e-commerce do Brasil. O Hiring Coders é uma parceria entre a GAMA Academy e a VTEX.
Dentro do programa, é necessário realizar semanalmente a entrega de vários "assignments", um dos assignments desta semana foi desenvolver uma landing page com algumas das melhores notícias sobre o COVID-19.

Construção desta aplicação

Esta é uma aplicação hard coded, ou seja, cada linha de código nesta aplicação foi escrita por mim, sem uso de nenhum framework que facilitasse o desenvolvimento, como o Bootstrap :( . Foi uma das regras para a entrega desse assignment.
Esta aplicação é feita puramente com HTML5 + CSS3.
